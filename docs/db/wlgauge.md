# 作业量表释义（仅供参考）
注释：周末及小长假期间，所有总耗时每休假一天增加25分钟。

## 极小
所有同学一般均可在校全部完成。        
总耗时应在5～25分钟范围内。       

## 小
一半同学可在校全部完成，其他同学离校后完成时间不超过35分钟。       
总耗时应在25～60分钟范围内。

## 正常
效率高的同学可在校全部完成，其他同学离校后完成时间不超过50分钟。       
总耗时应在60～85分钟范围内。       

## 大
更少的同学可在校全部完成，其他同学离校后完成时间不超过70分钟。       
总耗时应在85～110分钟范围内。       

## 极大
所有同学均难以在校全部完成。      
总耗时最多不得超过150分钟。
